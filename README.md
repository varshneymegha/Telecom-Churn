# Telecom-Churn

* Classifier to predict whether a customer will churn or not.
* Built many Classification Models with and without Dimensional Reductionality using PCA in Logistic Regression, Decision Trees, Random Forests, and XGBoost Classifiers.
Got the best Recall as `83.5% using Logistic Regression` and the `best Accuracy as 92.2% using XGBoost Classifier` along with PCA.
